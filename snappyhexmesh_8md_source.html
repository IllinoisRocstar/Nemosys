5 \section desc Description
6 
7 <em>snappyHexMesh</em> is a meshing engine that has been developed specifically for quality CFD hex(-dominant) mesh
8 generation. It is included with <em>OpenFOAM</em> and requires that the environment has been properly set. On a default
9 installation, this can be done by executing the following command in the working terminal:
10 
11  . /usr/lib/openfoam/openfoam2006/etc/bashrc
12 
13 Mesh generation with <em>snappyHexMesh</em> involves three steps:
14 1. <strong>Castellating</strong>: removes any cell beyond the meshing region of interest set by the user.
15 2. <strong>Snapping</strong>: moves the jagged edges of the cell along the boundary so that they are coincident with the boundary.
16 3. <strong>Layering</strong>: adds layers in the boundary region to ensure higher-quality cells in the transition region.
17 
18 The tutorial [available here](https://cfd.direct/openfoam/user-guide/v9-snappyHexMesh/) provides an excellent visualization
19 of how these three steps are applied to a 2D shape.
20 
21 <em>snappyHexMesh</em> requires a purely hexahedral base mesh, which can be created with another <em>Promesh</em> tool
22 such as <em>blockMesh</em>.

214  - <strong>`Axis Point 2`</strong>: An array of coordinates specifying the center of the top of a `searchableCylinder`. For example, [0, 0, 35]
215  - <strong>`Radius`</strong>: The radius of a `searchableCylinder` or a `searchableSphere`.
216  - <strong>`Center`</strong>: An array of coordinates specifying the center of a `searchableSphere`.

218 \subsection cast_mesh Castellated Mesh Controls
219 
220 Refinement in <em>snappyHexMesh</em> happens via splitting cells in half. For a 2 cm background mesh, the first level of
221 refinement will refine it to a 1 cm mesh. The second level of refinement will further refine it to a 0.5 cm mesh.
222 
223 Each level of refinement splits cells in half in each dimension. The general formula is \f$N(2^d)^R\f$, where \f$N\f$ is
224 the number of cells being split, \f$d\f$ is the number of dimensions (2 or 3), and \f$R\f$ is the levels of refinement
225 being applied. For example, in a 2D cell, the first level of refinement splits it into \f$(2^2)^1 = 4\f$ cells. The
226 fifth level of refinement would produce \f$(2^2)^5 = 1024 \f$ cells, each \f$2^{(-5)}\f$ the size. Similarly, for a 3D
227 cell, the first level of refinement splits it into \f$(2^3)^1 = 8\f$ cells. The fifth level of refinement would produce
228 \f$(2^3)^5 = 32768\f$ cells.

267 - <strong>`nFeatureSnapIter`</strong>: Number of iterations for feature snapping Optional; default is 10

270 - <strong>`multiRegionFeatureSnap`</strong>: If `true` (and `explicitFeatureSnap` is enabled), features between multiple surfaces will be captured. Optional; default is `false`

273 \subsection mesh_layer Mesh Layer Controls
274 
275 
276 - <strong>`relativeSizes`</strong>: Enables relative sizes option during layer addition. Allows the thickness of layers to be specified relative to the undistorted cell size rather than in absolute terms. Optional; default is `true`
277 - <strong>`expansionRatio`</strong>: Expansion ratio for layer addition. Optional; default is 1.3
278 - <strong>`finalLayerThickness`</strong>: Thickness (relative or absolute) of the layer furthest from the wall. Optional; default is 1.
279 - <strong>`minThickness`</strong>: Minimum overall thickness of the layers. Optional; default is 0.1
280 - <strong>`firstLayerThickness`</strong>: Thickness (relative or absolute) of the layer closest to the wall. Optional; default is -1.
281 - <strong>`thickness`</strong>: Overall thickness of the cells. Ignored if `finalLayerThickness` or `firstLayerThickness` are specified. Optional; default is -1.
282 - <strong>`nGrow`</strong>: Sets the number of layers that should not be grown if points are not extruded. Used to delay layer growth close to features. Optional; default is 0
283 - <strong>`featureAngle`</strong>: The angle in degrees above which surfaces are not extruded for layer addition. Optional; default is 30 degrees. If layers at corner edges are desired, set to 180
284 - <strong>`nRelaxIter`</strong>: Number of relaxation steps. Optional; default is 3
285 - <strong>`nSmoothSurfaceNormals`</strong>: Number of patch normal smoothing operations. Optional; default is 1
286 - <strong>`nSmoothNormals`</strong>: Number of smoothing iterations of interior mesh movement directions. Optional; default is 3
287 - <strong>`nSmoothThickness`</strong>: Smooth layer thickness over the surface patches. Optional; default is 2
288 - <strong>`maxFaceThicknessRatio`</strong>: Sets a maximum face to thickness ratio to stop layer growth on highly warped cells. Optional; default is 0.5
289 - <strong>`maxThicknessToMedialRatio`</strong>: Sets a maximum thickness to medial distance ratio to reduce layer growth, typically in narrow cavities. Optional; default is 1
290 - <strong>`minMedialAxisAngle`</strong>: Minimum angle to select the medial axis points. Optional; default is 90 degrees
291 - <strong>`nBufferCellsNoExtrude`</strong>: Sets the number of cells in a buffer region to gradually step down the number of layers. Optional; default is 0
292 - <strong>`nLayerIter`</strong>: Maximum number of layer addition iterations. Optional; default is 50
293 - <strong>`nRelaxedIter`</strong>: Number of iterations after which relaxed mesh quality controls are used. Optional; default is 20
294 - <strong>`slipFeatureAngle`</strong>: Allows the sliding of points on the patch without the layer grown if angle to the patch extrusion direction is larger than the one specified. Optional; default is 30 degrees
295 - <strong>`nMedialAxisIter`</strong>: Limits number of steps walking away from the surface. Optional; default is -1
296 - <strong>`nSmoothDisplacement`</strong>: Smoothing displacement of the mesh after the medial axis is determined. Optional; default is -1
297 - <strong>`Layers`</strong>: . Optional
298  - <strong>`Patch Name`</strong>: Name of the patch, defined in the `Geometry Definition` section.
299  - <strong>`nSurfaceLayers`</strong>: Number of layers per patch. Optional; default is 1
300  - <strong>`expansionRatio`</strong>: Expansion ratio. Optional; default is 1.
301  - <strong>`finalLayerThickness`</strong>: Thickness (relative or absolute) of the layer furthest from the wall. Optional; default is 1.
302  - <strong>`firstLyrThickness`</strong>: Thickness (relative or absolute) of the layer closest to the wall. Optional; default is -1.
303  - <strong>`thickness`</strong>: Overall thickness of the cells. Optional; default is -1
304  - <strong>`minThickness`</strong>: Minimum overall thickness of the layers. Optional; default is 1
305 
306 Only certain combinations of layer options can be used:
307 - `"expansionRatio"` and `"finalLayerThickness"`
308 - `"expansionRatio"` and `"firstLayerThickness"`
309 - `"thickness"` and `"finalLayerThickness"`
310 - `"thickness"` and `"firstLayerThickness"`
311 - `"expansionRatio"` and `"thickness"`

314 \subsection mesh_quality Mesh Quality Controls
315 
316 
317 - <strong>`maxNonOrtho`</strong>: Specifies the maximum allowable angle made by the vector between adjacent cell centers across the common face and face normal. Optional; default is 65
318 - <strong>`maxBoundarySkewness`</strong>: Sets the maximum allowable skewness in the boundaries. Optional; default is 20.
319 - <strong>`maxInternalSkewness`</strong>: Sets the maximum allowable internal skewness. Optional; default is 4.
320 - <strong>`maxConcave`</strong>: Sets the maximum concavity. Optional; default is 80.
321 - <strong>`minVol`</strong>: Sets the minimum cell volume in cubic meters. Optional; default is 1e-13
322 - <strong>`minTetQuality`</strong>: Sets the minimum tetrahedral element quality. Set to a small positive number (1e-15), it enables tracking. Set to a large negative number (-1e30) it enables optimal layer insertion. Optional; default is 1e-15
323 - <strong>`minArea`</strong>: Sets the minimum area for a cell face, in square meters. Optional; default is -1. (no minimum)
324 - <strong>`minTwist`</strong>: Sets the minimum value for twist. Optional; default is 0.02
325 - <strong>`minFaceWeight`</strong>: Sets the minimum face interpolation weight. Optional; default is 0.05
326 - <strong>`minVolRatio`</strong>: Sets the minimum volume ratio. Optional; default is 0.01
327 - <strong>`minDeterminant`</strong>: Sets the minimum cell determinant. Optional; default is 0.001
328 - <strong>`minTriangleTwist`</strong>: Sets the minimum triangle twist. Optional; default is -1. (no minimum)
329 - <strong>`qcnSmoothScale`</strong>: Smoothing iterations, used in conjunction with `errorReduction`. Optional; default is 5
330 - <strong>`errorReduction`</strong>: Error reduction, used in conjunction with `qcnSmoothScale`. Sets the amount to scale back displacement at points of error. Optional; default is 0.75
