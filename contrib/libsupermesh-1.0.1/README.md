## libsupermesh ##

libsupermesh parallel supermeshing library.

## Getting started ##

Documentation can be found in the libsupermesh manual. This can be built using:

```
#!sh

cd build
cmake .. -DENABLE_DOCS=1
make doc
```

The manual can then be found in build/doc/manual.pdf.
