How to use this tool:

This is a simple web application for transfering physical quantities form a source grid to a target grid. The application currently only support CGNS grids. You can download a sample source CGNS grid with physical quanitities from here and an empty target CGNS grid from here.

To use the tool, follow these steps:

Step1 - Click on "Upload Source Grid" to upload the source grid.

Step2 - Click on "Upload Target Grid" to upload the target grid.

Step3 - Click on "Transfer Solution" button to transfer physical quantities from source to target grid.

Step4 - Click on "Download" button to downlad the target grid with physical values.

Additional Features:

After source grid is loaded:

1- Click on "Grid statistics" button to see some of the statistics of the source grid.

2- Click on "Solution Names" button to see some available physical field defined on the source grid.
